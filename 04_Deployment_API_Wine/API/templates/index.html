API Documentation

Welcome to the API documentation for our wine quality prediction.

Endpoints

POST /prediction

This endpoint allows you to make a prediction of the salary amount (in dollars) according to the year.

How to request this endpoint?

You should request this endpoint using a POST method with a JSON content. Your JSON should contain at least "input" key with associated value of all the wine indicators in the format:

[[float]]

Example:
curl -i -H "Content-Type: application/json" -X POST -d '{ "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]] }' http://127.0.0.1:4000/predict

It returns a JSON like so: 

{"quality": [6]}

Also, if you wish, you can put multiple inputs at once:

Example:
curl -i -H "Content-Type: application/json" -X POST -d '{ "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8], [5.0, 0.98, 0.32, 18.9, 0.050, 75.0, 122.0, 0.401, 3.1, 0.21, 1.2]] }' http://127.0.0.1:4000/predict

errors

Our API will return an error with format: 

{
"code": HTTP status code,
"name": name of this error,
"description":a short description
}

Identified errors:

  • Missing JSON
  • Missing key "input"
